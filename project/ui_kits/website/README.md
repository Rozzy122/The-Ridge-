# The Ridge — Website UI Kit

A fun summer golf logbook. Four pages:
- **Home** — hero stats, recent rounds list, crew
- **Round Detail** — full scorecard with color-coded birdies/bogeys
- **Log Round** — form wizard
- **Courses** — courses we've played

## Components
- `Nav.jsx` — sticky green nav + `Pill`, `Avatar`, `Card`, `SectionTitle`
- `Home.jsx` — `HomePage`, `HeroStats`, `RoundRow`
- `Round.jsx` — `Scorecard`, `RoundDetailPage`, `LogRoundPage`, `CoursesPage`

Navigate by clicking the nav links, "Log a round", or any recent-round row.
